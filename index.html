Tentang Kami

Program Studi S1 Ilmu Informatika berada dibawah naungan Fakultas Teknik Universitas Katolik Darma Cendika (UKDC), dimana program studi ini terbentuk pada tanggal 31 Desember 2018 Rencana awal pembentukan program studi ini sebagai salah satu upaya UKDC untuk mengembangkan diri. Program Studi Ilmu Informatika Program Sarjana memiliki status ijin penyelenggaraan oleh Menteri Riset, Teknologi, dan Pendidikan Tinggi Nomor 120/KPT/I/2018.

Ilmu Informatika menawarkan 4 konsentrasi berdasarkan Curricula ACM (Association of Computing Machinery)
